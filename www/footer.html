Lineage nomenclature and model training:

For each Model a subset of training sequences were selected at random retaining 5/6 of the sequences in each class, providing the class had more than 7 sequences. Once the random forest was trained, we tested it with the remaining 1/5 of the corresponding sequences for each class. If the class had less than 7 sequences then all of them were used both in training and testing the corresponding model.

  • If you use the Rambaut classification model in your research, please also cite Rambaut et al., 2020. Please go to Rambaut supplementary data site for more information about the model.
  • If you use the Nexstrain classification model in your research, please also cite Hadfield et al., 2018. For more information please visit the Nextstrain website. Please go to Nextstrain supplementary data site for more information about the model.
